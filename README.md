\# AI-Powered Lecture Understanding System



This project converts lecture audio or YouTube videos into structured notes using AI.

It performs:

\- Speech-to-text transcription

\- Topic-wise summarization

\- Quiz question generation

\- PDF export of notes



\## Tech Stack

\- Python

\- Streamlit

\- Whisper

\- Transformers (T5, FLAN-T5)



\## How to Run

```bash

pip install -r requirements.txt

streamlit run app.py
